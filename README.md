# Criando-uma-Ferramenta-de-Controle-de-Investimentos-com-Excel
Ferramenta criada de acordo com aulas. A ferramenta faz parte do Desafio de Projeto



A Ferramenta final, a pasta no caso, foi desenvolvida conforme as instruções e ensinamentos do módulo DESAFIO DE PROJETO.
Todas as fórmulas ensinadas foram aplicadas nas duas planilhas que compõem o arquivo SIMULADOR INVESTIMENTOS FUNDOS IMOBILIARIOS EXCEL RESOLVIDO.XLSX

Nas aulas que antecedem o módulo DESAFIO DE PROJETO, foram apresentadas fórmulas como CONT.SE e CONT.SES.

Este repositório apresenta algumas imagens da ferramenta.
